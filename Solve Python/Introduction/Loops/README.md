# Loops

Read an integer N. For all non-negative integers i < N, print i*i. See the sample for details.

#### Sample Input
```python
5
```

#### Sample Output
```python
0
1
4
9
16
```

```bash
$ python <name>.py
```

```bash
$ pytest test_<name>.py

```

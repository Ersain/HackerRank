# Power - Mod power

You are given three integers: a, b, and m, respectively. Print two lines. 
The first line should print the result of pow(a,b). The second line should print the result of pow(a,b,m).

#### Sample Input
```python
3
4
5
```

#### Sample Output
```python
81
1
```

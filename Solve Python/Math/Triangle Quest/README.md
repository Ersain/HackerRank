# Triangle Quest
You are given a positive integer N. Print a numerical triangle of height N like the one below:
```
1
22
333
4444
55555
......
```

#### Input Format
A single line containing integer, N.

#### Sample Input
```python
5
```

#### Sample Output
```python
1
22
333
4444
```

#### Using with python
```bash
$ python <name>.py
```

```bash
$ pytest test_<name>.py

$ pytest test_<name>.py -v -s
```

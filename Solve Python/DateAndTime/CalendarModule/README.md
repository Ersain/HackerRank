# Calendar Module

You are given a date. Your task is to find what the day is on that date.

#### Sample Input
```
08 05 2015
```

#### Sample Output
```python
TUESDAY
```

#### Using with python
```bash
$ python <name>.py
```

```bash
$ pytest test_<name>.py
or
$ pytest test_<name>.py -v -s
```

# Arrays
You are given a space separated list of numbers. 
Your task is to print a reversed NumPy array with the element type float.


#### Input Format
A single line of input containing space separated numbers.

#### Sample Input
```python
1 2 3 4 -8 -10
```

#### Sample Output
```python
[-10.  -8.   4.   3.   2.   1.]
```

# Shape and Reshape

You are given a space separated list of nine integers. Your task is to convert this list into a 3x3 NumPy array.



#### Input Format
A single line of input containing 9 space separated integers.

#### Sample Input
```python
1 2 3 4 5 6 7 8 9
```

#### Sample Output
```python
[[1 2 3]
 [4 5 6]
 [7 8 9]]
```

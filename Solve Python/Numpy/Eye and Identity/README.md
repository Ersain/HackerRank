# Eye and Identity

Your task is to print an array of size NxM with its main diagonal elements as 1's and 0's everywhere else.



#### Input Format
A single line containing the space separated values of N and M. 

#### Sample Input
```python
3 3
```

#### Sample Output
```python
[[ 1.  0.  0.]
 [ 0.  1.  0.]
 [ 0.  0.  1.]]
```
